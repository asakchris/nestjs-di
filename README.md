## Description

[Nest](https://github.com/nestjs/nest) framework Dependency Injection examples.

Used following commands to create the projects and other resources:

```
nest new nestjs-di

nest g module computer
nest g module cpu
nest g module disk
nest g module power

nest g service cpu
nest g service power
nest g service disk

nest g controller computer
```

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```
